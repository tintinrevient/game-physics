# Fundamentals

## Kinematic Equations based on Constant Acceleration
* v2 = v1 + at
* v2<sup>2</sup> = v1<sup>2</sup> + 2a(s2 - s1)
* s2 = s1 + v1t + (at<sup>2</sup>)/2
